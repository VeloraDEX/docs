## Path parameters

<ParamField path="network" type="integer" required>
  Network ID. Matches the `network` from the source `priceRoute`.
</ParamField>

## Body parameters

<ParamField body="priceRoute" type="object" required>
  The `priceRoute` block returned by `GET /prices`, passed verbatim.
</ParamField>

<ParamField body="userAddress" type="string" required>
  End-user wallet address.
</ParamField>

<ParamField body="srcToken" type="string">
  Source token address. Matches `priceRoute.srcToken`.
</ParamField>

<ParamField body="destToken" type="string">
  Destination token address. Matches `priceRoute.destToken`.
</ParamField>

<ParamField body="srcAmount" type="string">
  Source amount in raw units. Required on `SELL` if not derived from `priceRoute`.
</ParamField>

<ParamField body="destAmount" type="string">
  Destination amount in raw units. Required on `BUY` if not derived from `priceRoute`.
</ParamField>

<ParamField body="slippage" type="integer">
  Slippage in basis points (e.g. `100` = 1%).
</ParamField>

## Response

Returns Augustus calldata plus transaction fields (`to`, `data`, `value`, `gasPrice`, `gas`, `chainId`).
