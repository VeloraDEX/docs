## Body parameters

<ParamField body="order" type="object" required>
  The order object produced by `POST /delta/orders/build`, passed verbatim.
</ParamField>

<ParamField body="signature" type="string" required>
  The EIP-712 signature of the order's typed data, hex-encoded.
</ParamField>

## Response

Order accepted; returns the order id and a status URL.
